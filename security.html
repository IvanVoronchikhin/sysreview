Производитель & История
Microsoft Corporation С Windows XP SP2 (2004) Microsoft Docs

Windows Firewall (официально — «Брандмауэр Защитника Windows») — хостовый межсетевой экран, встроенный в Windows начиная с XP SP2 в 2004 году. До этого Windows не имела собственного брандмауэра, что стало одной из причин массового распространения интернет-червей (Blaster, Sasser). Современная версия построена на платформе Windows Filtering Platform (WFP) — ядерной подсистеме фильтрации трафика, появившейся в Vista. Поддерживает три профиля: Домен, Частная сеть, Публичная сеть — с независимыми наборами правил для каждого.

Эксперименты (5 опытов)
1
Просмотр активных правил

Открыть wf.msc (Win+R → wf.msc). Изучить разделы «Inbound Rules» и «Outbound Rules». Найти правило для браузера (Chrome/Firefox). Просмотреть его свойства: на каком порту, протоколе, для какого профиля. Экспортировать политику в XML через Action → Export Policy.

2
Создание правила блокировки порта

Создать входящее правило, блокирующее TCP-порт 8080. Использовать мастер в wf.msc: New Rule → Port → TCP → 8080 → Block. Проверить результат через PowerShell: Test-NetConnection -Port 8080. Затем удалить правило и проверить снова. Сделать скриншоты до и после.

3
Включение журналирования

В свойствах профиля «Public» включить логирование: Logging → Log dropped packets → Yes. Запустить несколько сетевых приложений. Открыть лог C:\Windows\System32\LogFiles\Firewall\pfirewall.log. Проанализировать заблокированные соединения, найти подозрительные IP.

4
Управление через PowerShell

Вывести все активные правила: Get-NetFirewallRule | Where-Object {$_.Enabled -eq 'True'} | Select-Object DisplayName, Direction, Action. Создать правило командой New-NetFirewallRule -DisplayName "Test Block" -Direction Inbound -Protocol TCP -LocalPort 9999 -Action Block. Проверить и удалить.

5
Смена профиля и проверка поведения

Переключить сетевой адаптер между профилями «Private» и «Public» (Параметры → Сеть → свойства подключения). Сравнить, какие правила активны в каждом профиле. В публичном профиле по умолчанию заблокировано больше входящих соединений — зафиксировать разницу.

Вывод

Windows Defender Firewall — недооценённый инструмент. Большинство пользователей не осознают его возможностей, ограничиваясь стандартными настройками. При грамотной настройке он обеспечивает надёжный первый рубеж обороны без какого-либо снижения производительности. Интеграция с GPO делает его незаменимым в корпоративных средах. Для продвинутого управления — PowerShell-команды в разы эффективнее GUI.

Плюсы / Минусы
Плюсы
  • Встроен в Windows — нулевая стоимость
  • Нет нагрузки на производительность
  • Гибкое управление через PowerShell и GPO
  • IPsec интеграция из коробки
  • Три независимых сетевых профиля
  • WFP API для интеграции сторонних решений
Минусы
  • Сложный интерфейс wf.msc для новичков
  • Нет Deep Packet Inspection
  • Нет поведенческого анализа приложений
  • Нет уведомлений при попытках соединения
Производитель & История
Malwarebytes Inc., США Основана в 2008 г. malwarebytes.com

Malwarebytes основана в 2008 году Марцином Клечинским, который в 14 лет заразил семейный компьютер и не смог найти инструмент для очистки. Компания выросла из форумного инструмента RogueRemover до одного из самых популярных антивирусных решений с более чем 60 000 корпоративных клиентов. Бесплатная версия предлагает ручное сканирование; Premium — защиту в реальном времени, Anti-Ransomware, Anti-Exploit и веб-защиту. В 2016 году приобрела AdwCleaner; в 2021 — добавила модуль Browser Guard.

Эксперименты (5 опытов)
1
Быстрое сканирование и анализ результатов

Запустить Malwarebytes, выбрать «Quick Scan». Засечь время сканирования. Изучить отчёт: сколько объектов просканировано, найдены ли угрозы. Экспортировать отчёт в TXT. Обратить внимание на раздел PUP (Potentially Unwanted Programs) — они часто скрыты.

2
Тест с EICAR-файлом (безопасный тест-вирус)

Скачать стандартный EICAR test file с eicar.org (это безопасный текстовый файл, детектируемый как вирус). Проверить, обнаружит ли Malwarebytes файл при сохранении (защита в реальном времени) или при ручном сканировании. Зафиксировать время реакции.

3
Изучение карантина

Перейти в раздел «Quarantine». Если в системе были обнаружены угрозы — изучить их описание: тип, путь, хэш. Проверить один из заквантованных файлов на VirusTotal через «See details». Зафиксировать, согласны ли другие антивирусы с детекцией Malwarebytes.

4
Сравнение Free vs Premium возможностей

Изучить раздел «Real-Time Protection» (доступен в Premium). Сравнить список защитных модулей: Web Protection, Ransomware Protection, Exploit Protection. Составить таблицу: какие функции доступны бесплатно, какие — только в Premium ($44.99/год).

5
Полное сканирование и сравнение с Defender

Запустить полное сканирование Malwarebytes и одновременно — полное сканирование Windows Defender. Сравнить время работы и количество просканированных файлов. Если результаты различаются — определить, что один нашёл, а другой пропустил. Составить сравнительный отчёт.

Вывод

Malwarebytes занимает уникальную нишу: это не замена антивирусу, а его необходимое дополнение. Программа специализируется на угрозах, которые классические AV часто пропускают — PUP, adware, stalkerware, browser hijackers. Особенно ценен Anti-Ransomware модуль: он останавливает шифрование файлов даже от неизвестных ransomware-семейств через поведенческий анализ. Бесплатной версии достаточно для периодической ручной очистки системы.

Плюсы / Минусы
Плюсы
  • Работает совместно с любым антивирусом
  • Отличное обнаружение PUP и adware
  • Anti-Ransomware без базы сигнатур
  • Хорошая бесплатная версия
  • Современный понятный интерфейс
Минусы
  • Защита в реальном времени — только Premium
  • Высокое потребление RAM при сканировании
  • Иногда ложные срабатывания на легальный ПО
  • Нет встроенного брандмауэра
Производитель & История
Check Point Software Technologies Основан в 1998 г. (Zone Labs) zonealarm.com

ZoneAlarm — один из старейших персональных брандмауэров, разработанный компанией Zone Labs в 1998 году. В 2004 году Zone Labs была приобретена Check Point Software Technologies — крупнейшим в мире производителем сетевых межсетевых экранов. ZoneAlarm стал пионером концепции «application-level firewall» для домашних пользователей — первым решением, которое спрашивало разрешения у пользователя при попытке любого приложения выйти в интернет. В 2000-е годы был стандартным выбором для Windows XP без SP2. Сегодня продолжает развиваться, добавив Anti-Ransomware и Identity Protection.

Эксперименты (5 опытов)
1
Установка и первичная настройка зон

Установить ZoneAlarm Free Firewall. При первом запуске пройти мастер настройки. Изучить две зоны: Internet Zone (высокая защита) и Trusted Zone (доверенная сеть). Добавить домашнюю локальную сеть (192.168.x.x) в Trusted Zone. Зафиксировать применённые настройки скриншотом.

2
Проверка Stealth Mode через онлайн-сканер

Включить Stealth Mode в настройках ZoneAlarm. Зайти на сервис grc.com/shieldsup (ShieldsUP! от Gibson Research). Запустить «All Service Ports» сканирование. Зафиксировать, что все порты отображаются как «Stealth» (не «Closed»). Отключить ZoneAlarm и повторить — сравнить результат.

3
Контроль приложений: блокировка и разрешение

Открыть раздел «Application Control». Найти браузер в списке приложений. Сменить его разрешение с «Allow» на «Block». Попробовать открыть веб-страницу — убедиться, что она не открывается. Вернуть разрешение. Зафиксировать, что ZoneAlarm перехватил попытку соединения.

4
Анализ журнала соединений

Открыть ZoneAlarm Log Viewer (Tools → Log Viewer). Отфильтровать записи за последний час. Найти 3 приложения с наибольшим числом исходящих соединений. Определить, на какие IP/порты они подключались. Проверить один из IP-адресов через whois.domaintools.com.

5
Сравнение с Windows Defender Firewall

Составить сравнительную таблицу ZoneAlarm vs Windows Defender Firewall по критериям: интерактивные уведомления, application control, stealth mode, журналирование, управление через CLI, стоимость. Определить, в каких сценариях ZoneAlarm предпочтительнее встроенного брандмауэра Windows.

Вывод

ZoneAlarm остаётся актуальным именно благодаря интерактивному контролю приложений — возможности, которой нет у Windows Defender Firewall в удобном виде. Stealth Mode эффективно снижает видимость компьютера в интернете. Использование Check Point ThreatCloud обеспечивает корпоративный уровень базы угроз в бесплатном продукте. Оптимален для продвинутых пользователей, желающих видеть и контролировать каждое сетевое соединение приложений.

Плюсы / Минусы
Плюсы
  • Интерактивный контроль приложений
  • Stealth Mode — невидимость для сканеров
  • База угроз корпоративного уровня (Check Point)
  • Бесплатная версия достаточно функциональна
  • Многолетняя история и надёжность
Минусы
  • Может конфликтовать с Windows Defender Firewall
  • Требует регистрации даже для бесплатной версии
  • Устаревший интерфейс
  • Высокое число всплывающих уведомлений
  • Замедление загрузки системы
