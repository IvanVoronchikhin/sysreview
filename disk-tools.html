Disk Tools

Три ключевых инструмента для мониторинга состояния диска, глубокого тестирования и анализа использования файловой системы. Каждый раскрывается по клику.

Производитель & История
Crystal Dew World (Noriyuki Miyazaki) С 2006 года crystalmark.info

CrystalDiskInfo — бесплатная утилита для мониторинга состояния жёстких дисков и твердотельных накопителей через технологию S.M.A.R.T. (Self-Monitoring, Analysis and Reporting Technology). Разработана японским программистом Норийюки Мияцаки под псевдонимом «hiyohiyo». Первая версия вышла в 2006 году. Программа входит в состав большинства Live-CD/WinPE для диагностики систем. Поддерживает ATA, SATA, NVMe, USB (через SAT). Версия Shizuku Edition включает анимированного персонажа-интерфейс в стиле аниме.

Эксперименты (5 опытов)
1
Базовый S.M.A.R.T. мониторинг

Запустить CrystalDiskInfo, зафиксировать все атрибуты системного диска. Обратить внимание на Reallocated Sectors Count (05h), Pending Sectors (C5h), Uncorrectable Sectors (C6h). Записать текущие значения для сравнения в будущем.

2
Сравнение температур HDD vs SSD

При наличии обоих типов дисков — сравнить температуры в состоянии покоя и под нагрузкой (запустить тяжёлое копирование файлов). Зафиксировать разницу. Допустимый диапазон: HDD 30–50°C, SSD 30–70°C.

3
Power On Hours как индикатор износа

Просмотреть атрибут Power On Hours (09h). Рассчитать примерный возраст диска в годах. Сопоставить с гарантийным сроком производителя (обычно 3–5 лет для HDD, 5 лет для NVMe SSD). Оценить оставшийся ресурс.

4
Просмотр графика изменения атрибутов

В меню «Graph» выбрать атрибут Temperature и просмотреть его историю. Настроить автозапуск в трее и мониторинг с интервалом 5 минут. Через 30 минут работы — снять скриншот графика и проанализировать пики.

5
Экспорт и сохранение отчёта

Через меню Edit → Copy All выгрузить все данные S.M.A.R.T. в текстовый файл. Сравнить вывод с онлайн-базой BackBlaze S.M.A.R.T. Statistics. Составить письменный вывод о состоянии диска на основе полученных данных.

Вывод

CrystalDiskInfo — незаменимый инструмент первичной диагностики. Простой интерфейс скрывает глубокий анализ S.M.A.R.T. данных, позволяя мгновенно оценить состояние накопителя. Программа должна быть на каждой сервисной флешке системного администратора. Особенно ценна возможность фонового мониторинга с уведомлениями — это позволяет поймать деградацию диска до потери данных.

Плюсы / Минусы
Плюсы
  • Полностью бесплатна, без рекламы
  • Portable — запускается с флешки
  • Поддержка NVMe и USB-дисков
  • Фоновый мониторинг с уведомлениями
  • Активная разработка с 2006 года
  • Простой и понятный интерфейс
Минусы
  • Нет функции тестирования поверхности
  • USB-диски: данные могут быть неполными
  • Устаревший дизайн интерфейса
  • Некоторые NVMe атрибуты не расшифровываются
Производитель & История
Сергей Казанский, Украина С 1999 года (DOS-версия) hdd.by/victoria

Victoria — профессиональный инструмент глубокой диагностики жёстких дисков. Первая версия была написана ещё под DOS в 1999 году. Windows-версия появилась в 2000-х и стала стандартом для украинских и российских сервисных центров. Программа работает напрямую с диском через порты ввода-вывода (режим PIO/DMA), минуя файловую систему — это позволяет обнаруживать дефекты, невидимые ОС. Версия 5.x получила полностью переработанный интерфейс и поддержку NVMe.

Эксперименты (5 опытов)
1
Быстрое сканирование поверхности (Read)

Запустить Victoria, выбрать диск, установить режим «Read» (не деструктивный). Запустить тест с настройками по умолчанию. Наблюдать за картой секторов. Зафиксировать количество серых (медленных) и красных (ошибочных) секторов.

2
Анализ S.M.A.R.T. через Victoria

Перейти на вкладку S.M.A.R.T. в Victoria. Сравнить данные с выводом CrystalDiskInfo (эксперимент 1). Найти расхождения в интерпретации атрибутов. Особое внимание — значение Current/Worst/Threshold для атрибута 05h (Reallocated Sectors).

3
Измерение скорости по зонам (Speed map)

В разделе «Tests» запустить тест скорости чтения по зонам. Зафиксировать, как скорость HDD снижается от внешних дорожек (быстро) к внутренним (медленнее). Сравнить с SSD, где скорость стабильна по всему объёму.

4
Тест случайного доступа (Butterfly / Random)

Запустить режим случайного доступа. Замерить среднее время отклика на HDD и SSD. Ожидаемый результат: HDD — 8–15 мс, SATA SSD — 0.1–0.5 мс, NVMe — <0.05 мс. Зафиксировать данные и объяснить разницу.

5
Интерпретация карты дефектов

После сканирования изучить карту секторов. Записать статистику: сколько секторов <5мс (зелёных), <50мс (жёлтых), >150мс (красных). Если есть ошибочные сектора — определить их LBA-адрес и сопоставить с разделом файловой системы.

Вывод

Victoria — инструмент профессионального уровня. Она делает то, что не умеет ни одна другая бесплатная утилита: тестирует поверхность напрямую с аппаратным уровнем доступа. Незаменима при диагностике «умирающих» HDD. Режим Remap позволяет продлить жизнь диску, перенося данные из сбойных секторов в резервные. Однако требует понимания рисков: режим Write необратимо уничтожает данные на диске.

Плюсы / Минусы
Плюсы
  • Прямой доступ к диску — максимальная точность
  • Визуальная карта дефектов по секторам
  • Функция Remap сбойных секторов
  • Поддержка WinPE и Portable-режима
  • Бесплатна для некоммерческого использования
Минусы
  • Сложный интерфейс для новичков
  • Write-режим необратимо стирает данные
  • Требует прав администратора
  • Ограниченная поддержка NVMe vs SATA
Производитель & История
Antibody Software, Новая Зеландия С 2012 года diskanalyzer.com

WizTree — анализатор дискового пространства с уникальной технологией быстрого сканирования на основе прямого чтения Master File Table (MFT) файловой системы NTFS. Разработан новозеландской компанией Antibody Software. Первая версия вышла в 2012 году как альтернатива медленным аналогам (WinDirStat, TreeSize). Благодаря прямому обращению к MFT, WizTree анализирует диск объёмом 1 ТБ за 3–6 секунд, тогда как конкуренты тратят 5–15 минут. В 2021 году добавлена поддержка ReFS и расширенный экспорт.

Эксперименты (5 опытов)
1
Базовое сканирование и treemap

Запустить WizTree от администратора, выбрать системный диск C:. Засечь время сканирования. Изучить treemap: найти 5 крупнейших файлов. Обычно это hiberfil.sys (гибернация), pagefile.sys, крупные игры или видео. Записать их пути и размеры.

2
Поиск «мусора» — временные файлы

В фильтре расширений ввести «.tmp». Посмотреть суммарный объём временных файлов. Повторить для «.log», «.dmp» (дампы памяти). Зафиксировать найденное пространство. Сравнить с тем, что показывает стандартная «Очистка диска» Windows.

3
Анализ папки Windows и System32

Отфильтровать вид по папке C:\Windows. Определить, какие подпапки занимают больше всего места: WinSxS (хранилище компонентов), Installer, SoftwareDistribution (обновления). Записать размеры и сравнить с ожидаемыми (WinSxS обычно 5–15 ГБ).

4
Сравнение скорости: WizTree vs WinDirStat

Установить WinDirStat. Засечь время сканирования одного и того же диска обеими программами. Зафиксировать разницу в секундах. На диске 500 ГБ с 200K файлами WizTree должна быть в 30–100 раз быстрее. Сделать скриншоты итоговых времён.

5
Экспорт и анализ в CSV

Через File → Export нажать «Export All Files to CSV». Открыть файл в Excel/LibreOffice. Отсортировать по столбцу Size по убыванию. Построить сводную таблицу по расширениям: сколько занимают .exe, .dll, .mp4, .jpg. Составить краткий отчёт.

Вывод

WizTree решает классическую задачу «куда делось место на диске?» быстрее любого конкурента. Прямое чтение MFT — технически элегантное решение, которое делает программу практически мгновенной даже на 4-ТБ дисках. Treemap-визуализация интуитивна: достаточно одного взгляда, чтобы обнаружить «пожирателей» места. Необходима каждому системному администратору наряду с CrystalDiskInfo.

Плюсы / Минусы
Плюсы
  • Сканирование в 30–100× быстрее аналогов
  • Наглядная treemap-визуализация
  • Фильтрация по расширению и размеру
  • CSV-экспорт для внешнего анализа
  • Portable, не требует установки
  • Бесплатна для личного использования
Минусы
  • MFT-режим только для NTFS (не FAT32/exFAT)
  • Требует прав администратора для полного сканирования
  • Нет поддержки сетевых дисков в MFT-режиме
  • Платная версия для коммерческого использования
